# Keras implementation of PSPNet

### Requirements
    * Tensorflow (-gpu)
    * Keras
    * python3.5 and numpy
    * scipy <= 1.3.0
    * pillow

### Fine Tune
python finetune.py
47行：batch大小
48行：数据集路径
49行：模型读取和保存路径
54行：gpu数量
55行：使用的gpu的编号

### Train
python train.py
47行：batch大小
48行：数据集路径
49行：模型读取和保存路径
54行：gpu数量
55行：使用的gpu的编号
